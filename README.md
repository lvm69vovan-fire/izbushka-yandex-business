# izbushka-yandex-business

Генератор YML/XML-выгрузок для магазина [izbushka.kz](https://izbushka.kz) с подготовкой файлов для Яндекс.Бизнеса.

## Текущая версия

**V9_5_0**

## Возможности

- обработка списка карточек товаров;
- исключение товаров, которых нет в наличии;
- формирование общей YML/XML-выгрузки;
- разделение выгрузки по категориям;
- дополнительная корректировка категорий по названию товара;
- перенос элементов дымохода из разделов печей в правильную категорию;
- перенос печных комплектующих в отдельную категорию;
- журналирование выполнения;
- запуск в Windows через CMD;
- совместимость с Windows PowerShell 5.1, при этом PowerShell для запуска не требуется.

## Рабочая папка

Все входные и выходные данные рассчитаны на размещение в:

```text
D:\ЯндексБизнес
```

## Установка

1. Скопируйте содержимое репозитория в:

```text
D:\ЯндексБизнес\V9_5
```

2. Поместите список ссылок в файл:

```text
D:\ЯндексБизнес\izbushka-product-urls.txt
```

3. Запустите:

```text
RUN_V9_5.cmd
```

## Результаты

После успешного выполнения создаются:

```text
D:\ЯндексБизнес\izbushka-yandex-in-stock.xml
D:\ЯндексБизнес\izbushka-skipped.csv
D:\ЯндексБизнес\izbushka-yml-by-category.zip
D:\ЯндексБизнес\yml_by_category\
D:\ЯндексБизнес\v9-run-log.txt
```

## Требования

- Windows;
- Python 3.10 или новее;
- доступ к интернету;
- библиотеки из `requirements.txt`.

## Откат

Для отката сохраните предыдущую рабочую папку версии и запускайте соответствующий CMD-файл из неё. Результаты текущего запуска перед началом генерации очищаются.

## Проверенные сценарии

- корректный запуск без PowerShell;
- корректное отображение русского текста;
- обработка 1089 карточек;
- формирование 545 товаров в наличии;
- создание 24 категорий;
- корректный перенос дымоходов и комплектующих из категории банных печей.
